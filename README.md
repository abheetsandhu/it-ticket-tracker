# IT Ticket Tracker

## Overview
The IT Ticket Tracker is a beginner-friendly Python project that simulates a basic help desk ticketing system. It allows users to create, view, update, and summarize IT support tickets.

## Features
- Add new IT support tickets
- Track user name, issue, priority, and status
- View all tickets
- Update ticket status
- Generate a simple ticket summary

## Technologies Used
- Python
- Lists and dictionaries
- Functions
- Loops and conditionals

## Skills Demonstrated
- IT support workflow understanding
- Troubleshooting organization
- Ticket tracking
- Problem-solving
- Python programming basics
- Technical documentation
